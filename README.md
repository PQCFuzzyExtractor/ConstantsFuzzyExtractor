# ConstantsFuzzyExtractor

This repo contains a McEliece(348864f)-based code-offset fuzzy extractor implementation under the `fuzzy/` folder.

- Build/run instructions: see `fuzzy/README.md`
- Note: `fuzzy/liboqs.dll` is stored via **Git LFS**. After cloning, run `git lfs pull` so the real DLL is present.
